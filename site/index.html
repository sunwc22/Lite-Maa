Powerful Features

从启动到数据，一条线跑完

页面判断保持谨慎，识别结果保留证据，最终记录写入当前默认数据集位置。

01

启动唤醒

打开 MuMu、启动明日方舟，并穿过 START、开始唤醒和公告弹窗进入主页。

02

争锋频道采集

自动进入争锋频道、自娱自乐赛事，并按设置循环记录多轮对阵。

03

OCR 对阵识别

提取双方敌人名称、数量、支持按钮位置和最终礼物点数。

04

胜负判断

根据下注前后礼物数量变化推断获胜方，无法确认时保留 unknown。

05

JSONL 数据集

追加写入 data/duel_match_dataset.jsonl，保持每轮记录简洁可处理。

06

便携打包

支持轻量包和携带 PaddleOCR runtime 的便携包，方便迁移到目标电脑。

Automation Flow

每一轮都留下可追踪记录

  1. Start Wake

    进入明日方舟主页。

  2. Join Duel Event

    打开争锋频道并选择自娱自乐。

  3. OCR Extract

    读取敌人、数量、礼物和支持按钮。

  4. Support

    随机选择左侧或右侧队伍。

  5. Judge

    比较礼物变化，判定胜方。

  6. Append JSONL

    写入结构化对阵结果。

MaaLite is a local automation helper for personal Duel Channel data collection.
